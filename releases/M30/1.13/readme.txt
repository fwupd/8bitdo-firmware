Firmware update instruction
1. Press and hold L+R+START buttons on the controller to put it on its update mode. LED on the top will blink in red.
2. Run `sudo fwupdmgr install 8Bitdo-M30-1.13.cab`
3. Un-plug the gamepad when the upgrade process is done, then restart it before using.
